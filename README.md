1. Introduction / Objective
The Media & Entertainment Analytics Capstone Project focuses on analyzing user behavior and operational metrics in OTT platforms like Netflix. The primary goal is to leverage data analytics to understand user engagement, predict churn, recommend content, and optimize streaming efficiency.


Use Cases Overview
1. User Viewing Pattern Analysis – Identify trends and engagement patterns.
2. Content Recommendation System – Suggest content using behavioral and historical data.
3. Churn Prediction – Identify users likely to unsubscribe using predictive modeling.
4. Regional Content Demand Forecasting – Predict regional preferences using time-series analytics.
5. Peak Traffic and Load Optimization – Analyze traffic and load balancing strategies.
